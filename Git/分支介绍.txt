主分支：master分支

一、分支相关命令
查看本地分支：git branch 
查看本地分支和远程分支：git branch -a
创建本地分支：git branch 分支名
切换本地分支：git checkout 分支名
切换并创建本地分支：git checkout -b 分支名

删除分支：git branch -d 分支名
注意：删除分支前需要先切换到其他分支，需要删除的分支不能在使用中

合并分支：git merge 被合并的分支名
注意：1、当前分支为master，被合并分支为dev，执行git merge dev，会将dev分支的代码合并到master分支
2、修改其中一个分支提交修改，切换分支后是看不到提交记录的，合并分支后就可以在当前分支看到修改的记录

二、注意事项
1、所有分支操作都是在本地仓库执行的，不影响远程分支
2、所有的操作完成都需要git push之后才能修改远程仓库
